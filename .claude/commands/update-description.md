## 実行手順

1. ユーザーが指定したブログ記事ファイルを読み込む
2. frontmatterとマークダウン本文を解析
3. 最初の見出し(##)の次の段落テキストを抽出
   - 複数段落ある場合は全て結合
   - 次の見出しが出現するまでの全テキストを使用
4. 抽出したテキストでdescriptionフィールドを更新
5. **ユーザーの確認を求めずに直接実行**

## 注意事項

- frontmatterのtitleとdateは変更しない
- 最初の章の内容をそのままdescriptionに設定する(改変しない)
- 改行や空白は適切に処理する
